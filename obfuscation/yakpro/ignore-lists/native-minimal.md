# Native Minimal Ignore List

Current profile intentionally avoids most symbol renaming instead of maintaining a large ignore list.

Enabled transformations:

1. variable name obfuscation
2. string literal obfuscation
3. indentation stripping

Disabled transformations:

1. function renaming
2. class renaming
3. method renaming
4. property renaming
5. control-flow obfuscation
6. statement shuffling
